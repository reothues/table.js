use
```javascript
    $("#table").table();
```

with
```html
<table id="table" fixheight="150px" style="width:400px">
    <thead>
        <tr>
        <th>...</th>
        ...
        </tr>
    </thead> 
    <tr>
    ...
</table>
```

Set `fixheight="150px"` makes a scrollbar to the table, or you can set `pagesize="20"` to make a paging.
